BioPathNet

BioPathNet is an R shiny application that consists of multiple analysis “pods,” each of which analyzes transcriptomics data in a different way.

Pod 1 conducts Full Transcriptome Pathway Analysis utilizing Gene Set Enrichment Analysis (GSEA). This comprehensive analysis uses the entire transcriptome to determine which pathways and processes are upregulated or downregulated in the dataset, offering higher resolution and revealing subtle, wide-spread changes.

Pod 2 conducts Targeted Pathway Analysis utilizing Enrichr. This more traditional analysis offers a more focused view of the large pathway changes in the dataset.

Pod 3 conducts Perturbagen Analysis utilizing iLINCS. This analysis uses gene signatures to identify drugs that could treat or mimic the present condition, with the idea that genes and pathways that are the targets of these drugs likely play a key role in in the dataset.
